# 기능목록
- 주어진 횟수 동안 n대의 자동차는 전진 또는 멈출 수 있다.
- 사용자는 각 자동차의 이름을 부여하여 입력받으며 구분은 쉼표(,)로 한다.
- 자동차의 이름은 5자를 초과할 수 없다.
- 사용자는 몇 번의 게임을 진행할 것인지를 입력받는다.
- 전진하는 조건은 0에서 9사이에서 random 값을 구한 후 random 값이 4이상일 경우이다.
- 자동차의 상태를 화면에 출력한다. 어느 시점에서 출력할 것인지에 대한 제약은 없다.
